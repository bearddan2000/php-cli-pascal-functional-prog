# php-cli-pascal-functional-prog

## Description
Find the binomial coefficent.
Follows functional programming practices.

## Tech stack
- php 7.2.24

## Docker stack
- alpine:latest

## To run
`sudo ./install.sh -u`  

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
